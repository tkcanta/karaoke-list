# カラオケ曲リスト

カラオケに行った時に「俺何の曲歌えるんだっけ...」と悩む時間を解決するカラオケで歌える曲リストのサービス

サービスURL：[tk-production.xyz/karaoke-list](https://tk-production.xyz/karaoke-list)

## 機能

- 曲のカテゴリ分け（J-POP、アニメ、演歌）
- 曲の追加・編集・削除
- 曲名やアーティスト名での検索
- お気に入り曲の管理
- YouTubeリンクの保存と再生
- キー設定の保存

## 使用技術

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Web Storage API
